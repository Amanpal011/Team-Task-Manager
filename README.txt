Team Task Manager

A full-stack task management web application built using React, TypeScript, Node.js, Express.js, and MongoDB.

Features:
- User Authentication
- Task Management
- Team Collaboration
- Project Tracking
- Responsive UI

Tech Stack:
Frontend: React, TypeScript, Tailwind CSS
Backend: Node.js, Express.js
Database: MongoDB
